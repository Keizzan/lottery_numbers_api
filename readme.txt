Project Name: 
Random Numbers

Synopsis:
Program contain API which generate 5 random numbers and VueJS as frontend to display them

Application requirement: 
Apache/2.4.41 
PHP Version 7.4.3
Vuejs:
"core-js": "^3.6.5",
"vue": "^3.0.0"


For Developers
-----------------
    1. Run npm install to install VueJS dependencies
    2. Run npm run serve to run development server
    3. Run npm build to prepare for production
